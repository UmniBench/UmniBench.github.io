## 快速部署

### 方案一：独立仓库（推荐）

1. 在 GitHub 创建新仓库：`UmniBench.github.io`

2. 将 docs 内容复制到仓库根目录：
```bash
git clone https://github.com/UmniBench/UmniBench.github.io.git
cd UmniBench.github.io
cp -r /path/to/UmniBench/docs/* .
git add .
git commit -m "Initial website"
git push
```

3. 网站将自动部署到：`https://umnibench.github.io/`

### 方案二：使用 docs 文件夹

1. 提交 docs 文件夹到现有仓库：
```bash
cd /home/leyangchen/data2/UmniBench
git add docs/
git commit -m "Add GitHub Pages website"
git push origin main
```

2. 在 GitHub 仓库设置中：
   - 进入 Settings > Pages
   - Source 选择 "Deploy from a branch"
   - 选择 `main` 分支和 `/docs` 文件夹
   - 保存

3. 网站将部署到：`https://yourusername.github.io/UmniBench/`

## 本地测试

部署前先在本地测试：

```bash
cd docs
python -m http.server 8000
```

然后访问：http://localhost:8000
